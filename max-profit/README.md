# Max Profit Problem

This project calculates the maximum profit that can be earned within a given time limit using Node.js.

Based on the available time, different combinations of properties are evaluated. For each valid combination, the total construction time is checked and profit is calculated using the remaining operational time after each build.

The program:
- Tries all possible valid build combinations
- Calculates total earnings for each case
- Selects the combination that gives the highest profit
- Displays the final mix along with the maximum profit

The implementation is written in plain Node.js without external libraries.

How to run:
1. Install dependencies (if any)
   npm install

2. Run the program
   npm start
